# Tridev Koneti — Portfolio

Personal resume and portfolio site for Tridev Koneti, Senior Data Analyst & BI Developer.

**Live site:** https://tridev-koneti.github.io/

## About

A single-page site covering profile, career highlights, work experience, skills, and education, plus links to GitHub, Medium, and LinkedIn.

## Stack

Plain HTML, CSS, and JavaScript — no build step, no dependencies. Everything lives in `index.html`.

- Light/dark theme toggle (dark by default)
- Responsive layout with a collapsible mobile nav
- Scroll-spy navigation highlighting
- Scroll-triggered reveal animations

## Editing

1. Edit `index.html` directly.
2. Commit and push to `main`:
   ```
   git add index.html
   git commit -m "Update site"
   git push
   ```
3. GitHub Pages rebuilds automatically, usually within a minute or two.

## Deployment

Hosted via [GitHub Pages](https://pages.github.com/), served from the `main` branch at the repository root.
